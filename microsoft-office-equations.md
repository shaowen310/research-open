# Microsoft Office Equations

## Insert an Equation

`alt` + `+` 

## Math Symbols

| Symbol | Code |
| :--- | :--- |
| $$\mathbb{R}$$, $$\mathbb{Z}$$ | `\doubleR` `\doubleZ` |
| $$\mathcal{L}$$ | `\scriptL` |
| $$\times$$,$$\cdot$$,$$\circ$$ | `\times` `\cdot` `\circ` |
| $$\arg \underset{\phi,T}{\max}$$ | `"arg"` then find "max" in "Limit and Log" tab |
